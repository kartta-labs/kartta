Warper Help

Uploading

To upload a new map to {{ SITE_NAME }} warper, click the Upload link near the top right corner of the page. Add a title for the map, and give values for as many of the optional fields as possible. Choose a local image file to upload, or give the URL of an image to be imported, and click the Create button.

The map that you upload or import must be a raster image format such as png, tiff, or jpeg. PDFs are not accepted; if you have a map in PDF format, convert it to an image before uploading.

Rectifying

To georectify an uploaded map, first go to the map's main page by clicking on its title on the home page or on the Browse page. On the map's main page, click the Rectify tab and use the two views to place several (at least three, and the more the better) pairs of control points. When you have finished adding control points, click the 'Warp Image' button.

You can return to the map to add more control points, and/or to edit previously added control points, at any time. Each time you edit, delete, or add control points, click the 'Warp Image' button to update the map.

Viewing and Exporting

You can view a rectified map by clicking on the Preview tab on the map's main page.

The Export tab on the map's main page has links to let you download the rectified map in various digital formats.

Deleting a Map

To delete a map, use the Delete this map link at the top of the Edit tab on the map's main page. Note that all maps in {{ SITE_NAME }} warper are public; deleting a map will cause it to be deleted from the {{ SITE_NAME }} warper servers and will prevent anyone from downloading the map from that point on, but it will not delete any copies that had previously been downloaded. See the FAQ for more information about what data is deleted when you delete a map.

Deleting Your {{ SITE_NAME }} Account

To delete your {{ SITE_NAME }} account, contact us at {{ CONTACT_EMAIL }}. Note that deleting your {{ SITE_NAME }} account will not delete your Google account -- it just deletes your Google account's association with {{ SITE_NAME }}. This will delete all maps that you have uploaded to {{ SITE_NAME }} warper; it will not delete contriutions you have made to Editor, but it will dissociate them from your account. See the FAQ for more details.
